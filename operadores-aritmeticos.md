## Operadores aritmeticos

- suma
```python
5+5
```


- multiplicacion
```python
5 * 4 * 3 * 2 * 1
```


- division
```python
  10 / 5
  20 / 3
```

- division con residuo solo la parte entera
```python
  20 // 3
```


- division con residuo solo la parte del residuo
```python
  20 % 3
```


- potencia
```python
  2 ** 2
```

- raices, se debe importar la lib **math**
 ```python
   import math
   raiz = math.sqrt
```
