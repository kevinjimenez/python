## Operadores logicos

### - Operador logico AND
##### En python a diferencia de otros lenguajes se usa la palabra **and** en vez de **&&**
```python
es_estudiante = True
## True
trabaja = False
##False
es_estudiante and trabaja
## False
```

### - Operador logico OR
##### En python a diferencia de otros lenguajes se usa la palabra **or** en vez de **||**
```python
es_estudiante = True
## True
trabaja = False
##False
es_estudiante or trabaja
## True
```

### - Operador logico NOT
##### En python a diferencia de otros lenguajes se usa la palabra **not** en vez de **!**
```python
es_estudiante = True
## True
not es_estudiante 
## False
trabaja = False
## True
not trabaja
## True
```

### - Operador de comparacion
```python
numero1 = 5
## 5
numero2 = 5
## 5
numero1 == numero2
## True

numero3 = 6
## 6
numero4 = 7
## 7
numero3 == numero4
## False
```

### - Operador de distinto
```python
numero1 = 5
## 5
numero2 = 5
## 5
numero1 != numero2
## False

numero3 = 6
## 6
numero4 = 7
## 7
numero3 != numero4
## True
```
