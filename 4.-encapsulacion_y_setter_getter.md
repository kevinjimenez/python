## Encapsulacion

- agrupar datos y comportamiento
- control acceso a datos
- provie modificacione no autorizadas

```python
class Coordenada:

    def __init__(self, valor = 0):
        self._valor = valor


    @property
    def get_x():
        return self._valor

    @get_x.setter
    def set_x(valor):
        self._valor = valor 

```
